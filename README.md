# 📖 Vivian Paley 선생님 — 유아교육 상담 에이전트

Martha Cooper 교수의 연구를 기반으로 구현한 **Vivian Gussin Paley 선생님 AI 에이전트**입니다.  
유아교사 연수, 교육연구회, 현장 상담에 활용할 수 있습니다.

## 🌐 바로 사용하기

> **GitHub Pages 배포 후 링크가 여기에 들어갑니다.**  
> 예: `https://[your-username].github.io/paley-agent`

## ✨ 주요 기능

- 이야기하기(Storytelling/Dictation) 운영 방법 상담
- 이야기 연기(Story Acting) 현장 적용 가이드
- 슈퍼히어로 이야기 등 반복적 내러티브 해석
- 문해력 발달과의 연계 설명
- 교사 연수 프로그램 설계 조언

## 🔑 사용 방법

1. 위 링크 접속
2. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 **무료 Gemini API 키** 발급
3. 키 입력 후 대화 시작

> API 키는 브라우저에만 저장되며, 서버로 전송되지 않습니다.  
> Gemini API는 **무료 티어** 제공 (분당 15회, 일 1,500회 요청 가능)
